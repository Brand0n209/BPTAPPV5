Bright Prodigy API

Welcome to the Bright Prodigy API server. This API provides access to calendar services and more.

Health Check

GET /health

Check the health status of the API.

{"status":"OK","message":"Server is healthy"}

Calendar API

GET /api/calendar/:id/events

Get events from a calendar.

Available calendar IDs: greetings, brightProdigy, crew1, pending

Optional query parameters:

  • timeMin - Start date (ISO string)
  • timeMax - End date (ISO string)
  • maxResults - Maximum number of events to return
POST /api/calendar/:id/event

Create a new event in a calendar.

Available calendar IDs: greetings, brightProdigy, crew1, pending

Example request body:

{
  "summary": "Team Meeting",
  "description": "Weekly team status update",
  "start": {
    "dateTime": "2023-06-15T10:00:00-07:00",
    "timeZone": "America/Los_Angeles"
  },
  "end": {
    "dateTime": "2023-06-15T11:00:00-07:00",
    "timeZone": "America/Los_Angeles"
  },
  "attendees": [
    {
      "email": "team@brightprodigy.com",
      "displayName": "Bright Prodigy Team"
    }
  ]
}
DELETE /api/calendar/:id/event/:eventId

Delete an event from a calendar.

Available calendar IDs: greetings, brightProdigy, crew1, pending
